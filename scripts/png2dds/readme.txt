Easy2Convert PNG to DDS
Version 2.9, Freeware
https://www.easy2convert.com/png2dds/

Introduction
------------
Easy2Convert PNG to DDS is a small freeware 
utility for converting Portable Network Graphics files (.png) 
to DirectDraw Surface files (.dds) easily and quickly. 

System requirements
-------------------
 * Windows 2000/XP/2003/Vista/7/8
 * less than 3 Mb of free space on hard disk
 
Installation
------------
Run an installer/unzip all files to any folder and run 
the executable file (usually, "png2dds.exe").

Getting started
---------------
1. Select a png-file to convert
2. Select an output dds-file
3. Click on the "Convert" button.
